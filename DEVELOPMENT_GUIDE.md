# 深圳市鸟类分布地图小程序 - 开发指南

## 项目概述

这是一个展示深圳市鸟类分布的微信小程序MVP版本，用户可以通过交互式地图查看深圳不同区域的鸟类分布情况，并获得鸟类的详细信息。

## 环境准备

### 1. 微信开发者工具
- 下载并安装最新版微信开发者工具
- 登录微信开发者账号

### 2. 项目配置
- 将此项目导入微信开发者工具
- 确保项目配置中的 AppID 正确 (已在 project.config.json 中配置)

### 3. 云开发环境
- 在微信公众平台开通云开发功能
- 在项目中开通云开发服务

## 数据库设计

### 鸟类观察集合 (bird_observations)
- `_id`: 字符串 - 记录唯一标识
- `species`: 字符串 - 物种名称
- `scientificName`: 字符串 - 学名
- `commonName`: 字符串 - 通用名
- `location`: 对象 - 经纬度信息 {latitude, longitude}
- `locationName`: 字符串 - 位置名称
- `date`: 字符串 - 观察日期
- `time`: 字符串 - 观察时间
- `habitat`: 字符串 - 栖息地类型
- `season`: 字符串 - 季节
- `description`: 字符串 - 描述信息
- `imageUrl`: 字符串 - 图片链接
- `audioUrl`: 字符串 - 音频链接（可选）
- `observer`: 字符串 - 观察者
- `confidence`: 数字 - 观察可信度（0-1）
- `verified`: 布尔值 - 是否已验证
- `region`: 字符串 - 区域（固定为"深圳"）
- `_openid`: 字符串 - 提交用户ID
- `createdAt`: 日期 - 创建时间
- `updatedAt`: 日期 - 更新时间

## 云函数说明

### getBirdData
- 功能：获取鸟类数据
- 输入：{ location: { latitude, longitude }, radius: number }
- 输出：{ success: boolean, data: array }

### addBirdData
- 功能：添加鸟类数据
- 输入：{ birds: array }
- 输出：{ success: boolean, data: object }

## 部署说明

1. 在云开发控制台创建名为 `birds` 的数据库集合
2. 部署云函数 `getBirdData` 和 `addBirdData`
3. 上传必要的静态资源（图片、音频）
4. 在微信公众平台提交审核

## 注意事项

- 遵守微信小程序开发规范
- 图片资源需要压缩以减少加载时间
- 音频文件需注意版权问题
- 用户提交的内容需要审核机制

## 调试技巧

- 使用 console.log 输出调试信息
- 在开发者工具中使用调试器断点
- 检查网络请求是否正常
- 验证云函数返回数据格式
