# 深圳市鸟类分布地图小程序

## 项目结构

```
miniprogram/
├── app.js          # 小程序逻辑
├── app.json        # 小程序公共配置
├── app.wxss        # 小程序公共样式
├── pages/          # 页面目录
│   └── index/      # 首页
│       ├── index.js    # 首页逻辑
│       ├── index.json  # 首页配置
│       ├── index.wxml  # 首页结构
│       └── index.wxss  # 首页样式
├── images/         # 图片资源
└── sitemap.json    # 站点地图配置
```

## 功能说明

这是一个展示深圳市鸟类分布的微信小程序MVP版本，主要功能包括：

1. 交互式地图展示深圳市真实鸟类分布
2. 基于实际观察数据的标记展示
3. 点击地图标记查看鸟类详细信息和观察详情
4. 支持按位置检索附近的鸟类观察记录
5. 数据验证和深圳区域边界检查

## 开发说明

1. 使用原生微信小程序框架开发
2. 集成了地图组件展示鸟类分布
3. 实现了从云端获取真实鸟类观察数据的功能
4. 支持按地理位置检索鸟类数据
5. 包含数据验证和位置校验机制

## 使用方法

1. 在微信开发者工具中打开此项目
2. 确保已开通云开发环境
3. **重要**: 首次使用前请先部署云函数（详见 `CLOUD_DEPLOY_GUIDE.md`）
4. 编译并预览小程序

## 云函数部署

本项目包含多个云函数，需要在微信开发者工具中部署后才能正常使用。请参考 `CLOUD_DEPLOY_GUIDE.md` 文档了解详细部署步骤。
