# 深圳鸟类分布地图 - 使用说明

## 数据来源说明

当前系统正在使用本地存储的真实鸟类观察数据。要启用云端数据功能，请按照以下步骤操作：

### 部署云函数

1. 打开微信开发者工具
2. 在 `cloudfunctions` 目录下找到所有云函数
3. 右键点击每个云函数并选择"上传并部署"
4. 确保所有云函数都成功部署到云端

### 云端功能优势

- 实时更新的鸟类观察数据
- 用户贡献的新观察记录
- 深圳各区域最新的鸟类分布信息
- 更丰富的多媒体内容

### 本地数据功能

即使未部署云端函数，您仍可体验完整功能：

- 查看深圳代表性观鸟点的鸟类分布
- 详细了解各种鸟类的信息
- 使用地图查看鸟类观察位置
- 所有观鸟点和鸟类数据均为真实信息
