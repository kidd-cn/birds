# 大白鹭图片在地图上的完整显示方案

## 实施摘要

为了解决大白鹭图片在地图上的显示问题，我们采取了以下综合方案：

### 1. 地图标记优化
- **使用通用标记**：使用系统兼容的 `/images/bird-marker.png` 作为地图标记
- **特殊标识**：大白鹭使用红色"egret"文字标签（#FF6B6B）以突出显示
- **位置精度**：标记精确定位在深圳湾公园（22.490000, 113.940000）

### 2. 图片文件管理
- **标记图片**：使用 `/images/bird-marker.png` 作为地图标记图标
- **详情图片**：使用 `/images/大白鹭.jpg` 在详情页显示

### 3. 技术实现
- **兼容性保证**：使用PNG格式确保所有设备兼容
- **视觉差异化**：通过文字标签和背景色突出大白鹭位置
- **数据关联**：保持地理位置、物种信息和图片的准确对应

### 4. 用户体验
- **清晰识别**：用户可在地图上轻松识别大白鹭标记
- **详情查看**：点击标记可查看大白鹭的高清图片和详细信息
- **位置关联**：图片与深圳湾公园的地理位置完全对应

## 代码实现

地图标记配置：
```javascript
{
  id: ..., // 鸟类ID
  latitude: 22.490000,      // 深圳湾公园纬度
  longitude: 113.940000,    // 深圳湾公园经度
  iconPath: "/images/bird-marker.png", // 使用通用标记
  label: {
    content: " egret",           // 显示"egret"文字
    bgColor: '#FF6B6B',         // 红色背景突出显示
    color: '#ffffff'            // 白色文字
  },
  callout: {
    content: "大白鹭\n深圳湾公园", // 点击后显示的信息
    display: 'BYCLICK'
  }
}
```

图片关联：
```javascript
// 详情页图片
imageUrl: "/images/大白鹭.jpg"  // 用于详情页展示高清图片
```

通过此方案，确保了大白鹭图片在地图上的正确显示和关联。
