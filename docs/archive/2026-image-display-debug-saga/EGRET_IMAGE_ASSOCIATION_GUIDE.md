# 大白鹭图片地图关联配置指南

## 目标
将 images/大白鹭.jpg 图片正确关联到地图上的深圳湾公园位置

## 实施步骤

### 1. 图片规格要求
- 文件路径: /miniprogram/images/大白鹭.jpg
- 格式: JPG (已确认)
- 建议尺寸: 32x32 到 64x64 像素
- 文件大小: 小于40KB

### 2. 代码配置
在地图标记中正确配置大白鹭的图片引用：

```javascript
// 大白鹭的标记配置
{
  id: 2,                                    // 大白鹭数据ID
  latitude: 22.490000,                     // 深圳湾公园纬度
  longitude: 113.940000,                   // 深圳湾公园经度
  iconPath: "/images/大白鹭.jpg",           // 正确的图片路径
  width: 30,                               // 标记宽度
  height: 30,                              // 标记高度
  callout: {                               // 点击弹出信息
    content: "大白鹭\n深圳湾公园",
    display: 'BYCLICK'
  }
}
```

### 3. 路径规范
- 使用绝对路径: `/images/大白鹭.jpg` (以"/"开头)
- 确保图片文件存在于小程序代码包中
- 避免使用中文字符或特殊符号（尽管微信小程序支持）

### 4. 验证方法
- 检查图片文件是否在正确路径
- 确认图片格式兼容性
- 在真机上测试标记显示效果

### 5. 故障排除
如果图片仍未显示：
- 检查开发者工具控制台是否有错误信息
- 确认图片文件大小和格式
- 临时使用较小尺寸的图片进行测试
- 尝试使用英文命名的图片文件

通过以上配置，大白鹭的图片应当能正确显示在地图上的深圳湾公园位置。
