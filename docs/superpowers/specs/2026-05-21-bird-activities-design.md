# 首页趣味活动功能设计文档

## 1. 功能概述

本功能旨在为"深圳观鸟助手"小程序的首页增加一个趣味活动模块，通过游戏化的方式提高用户参与度和对鸟类知识的兴趣。该模块包含三种互动类型：鸟类识别游戏、鸟类知识问答和观鸟打卡活动。

## 2. 功能结构

### 2.1 入口设计
- 在首页(index.wxml)地图上方增加"趣味活动"入口按钮
- 点击按钮弹出活动选择界面
- 选择界面包含三个选项卡：识别游戏、知识问答、打卡活动

### 2.2 活动类型详述

#### A. 鸟类识别游戏 (Identification Game)
- **难度分级**：5级（新手、初级、中级、高级、专家）
- **内容来源**：使用项目中真实鸟类图片数据
- **玩法机制**：
  - 显示一张鸟类图片
  - 提供4个选项（1个正确，3个干扰项）
  - 难度影响图片质量或清晰度
  - 记录答对题数和得分
- **数据要求**：每题需包含鸟类的准确学名、中文名、特征描述
- **反馈机制**：答对/答错提示，解释正确答案

#### B. 鸟类知识问答 (Knowledge Quiz)
- **题目数量**：5道题
- **总分**：100分（每题20分）
- **题目类型**：单选题
- **内容范围**：鸟类外貌特征为主
- **评分标准**：
  - 90-100分：鸟类专家
  - 80-89分：观鸟达人
  - 70-79分：鸟类爱好者
  - 60-69分：入门观鸟者
  - 60分以下：观鸟新手
- **反馈机制**：答题结果、总体评价

#### C. 观鸟打卡活动 (Check-in Activities)
- **打卡机制**：完成特定观鸟点的虚拟打卡任务
- **奖励系统**：完成打卡获得该景点专属徽章
- **徽章类型**：
  - 深圳湾公园徽章
  - 福田红树林生态公园徽章
  - 华侨城国家湿地公园徽章
  - 内伶仃福田国家级自然保护区徽章
  - 仙湖植物园徽章
- **展示方式**：个人中心徽章墙

## 3. 界面设计

### 3.1 入口按钮
- 位置：地图上方中央
- 样式：圆形按钮，使用活动图标
- 文案："趣味活动"

### 3.2 活动选择界面
- 类型：模态弹窗
- 布局：三列网格布局
- 每个项目：图标 + 标题 + 描述

### 3.3 游戏界面
- 鸟类识别游戏：图片展示区 + 选项区 + 计分板
- 知识问答：题目展示区 + 选项区 + 进度条
- 打卡活动：景点列表 + 徽章展示区

## 6. 实现要点

### 6.1 性能考虑
- 图片懒加载，避免一次性加载过多资源
- 游戏状态本地缓存，保证用户体验连续性

### 6.2 用户体验
- 游戏过程流畅，动画效果自然
- 结果反馈及时，鼓励用户参与
- 界面设计与整体风格保持一致

### 6.3 数据准确性
- 题目内容由真实鸟类数据生成
- 物种信息科学准确，避免误导用户

## 7. 测试要点

- 活动入口按钮点击正常
- 三种活动类型均能正常运行
- 游戏逻辑正确（计分、难度等）
- 数据持久化（分数、徽章等）
- 界面适配不同屏幕尺寸

## 8. 安全与兼容性

- 输入验证，防止恶意数据注入
- 错误处理，避免应用崩溃
- 兼容不同版本微信小程序
