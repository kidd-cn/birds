# 观鸟学堂模块设计文档

## 一、模块概述

**模块名称：** 观鸟学堂

**定位：** 引导用户学习与自然相处的正确价值观，通过闯关式互动体验让用户内化观鸟礼仪、安全知识和进阶技能。

**核心价值：** 培养负责任的观鸟者，传递尊重自然、保护环境的理念。

---

## 二、入口设计

- **位置：** 首页底部Tab栏新增"观鸟学堂"按钮
- **图标：** 手绘羽毛/书本组合图标
- **样式：** 与"发现"Tab并列，图标+文字"观鸟学堂"

---

## 三、整体架构

### 3.1 页面结构

1. **学堂首页（地图视图）** — 4个主题站点的闯关地图
2. **主题详情页** — 站点介绍 + 该主题所有关卡列表
3. **关卡互动页** — 每关的互动任务界面
4. **成就页面** — 与每日签到"查看成就"合并

### 3.2 导航流程

```
首页 → 观鸟学堂入口 → 地图视图 → 选择站点 → 关卡列表 → 互动答题 → 通关 → 下一关
                                              ↓
                                    全部完成 → 主题徽章解锁 → 海报生成
```

---

## 四、地图界面设计

### 4.1 视觉风格

- **背景：** 手绘森林/湿地插画风格，温暖亲近自然
- **站点图标：**
  - 装备篇：望远镜 🔭
  - 安全篇：盾牌 🛡️
  - 行为篇：树叶 🍃
  - 进阶篇：羽毛 ✈️
- **装饰：** 手绘风格的路径连接各站点

### 4.2 站点状态

| 状态 | 视觉表现 |
|------|----------|
| 未解锁 | 灰色图标 + 🔒 锁 |
| 已解锁 | 金色发光 + ⭐ |
| 已完成 | 打勾标记 + 该主题徽章 |

### 4.3 交互设计

- 点击站点 → 弹出主题介绍面板
- 面板内展示：该主题简介 + 关卡列表（子关卡完成进度）
- 点击"进入主题" → 进入主题详情页

---

## 五、关卡内容设计

### 5.1 主题与关卡结构

**装备篇（3关）**
- 关卡1：选择合适的望远镜
- 关卡2：相机与镜头配置
- 关卡3：穿着与随身装备

**安全篇（3关）**
- 关卡1：天气与环境判断
- 关卡2：安全出行要点
- 关卡3：紧急情况应对

**行为篇（3关）**
- 关卡1：保持安静与距离
- 关卡2：不干扰鸟类生活
- 关卡3：保护环境与垃圾分类

**进阶篇（3关）**
- 关卡1：科学记录方法
- 关卡2：知识分享规范
- 关卡3：团队协作观鸟

**总计：4主题 × 3关 = 12关**

### 5.2 每关结构

1. **封面：** 写实摄影图 + 关卡标题
2. **引导语：** 手绘旁白框展示
3. **互动题目：** 3-5道题
4. **完成页：** 徽章解锁动画 + 进入下一关按钮

### 5.3 题型设计（混合型）

- **选择题：** 展示图片 + 正确/错误做法选项
- **排序题：** 拖拽卡片排列正确顺序
- **补全题：** 图片 + 缺失关键词填空

### 5.4 线性解锁

- 必须完成前一个关卡才能解锁下一个
- 站点内关卡按顺序解锁
- 未解锁关卡显示灰色 + 🔒

---

## 六、成就系统设计

### 6.1 徽章墙

**主题徽章（4个）：**
- 装备达人
- 安全卫士
- 观鸟使者
- 进阶专家

**关卡徽章（12个）：**
- 每关完成后解锁对应徽章

**特殊徽章：**
- 观鸟学徒（完成第1关）
- 观鸟学者（完成全部12关）
- 连续登录（连续7天学习）

### 6.2 社交分享

**海报生成：**
- 用户选择已获得徽章组合
- 生成个性化成就海报
- 可分享到朋友圈

**排行榜：**
- 积分维度：完成关卡获得积分
- 三个维度：本周、本月、全部

### 6.3 页面入口

- 成就页面与每日签到"查看成就"合并
- 入口在每日签到页面

---

## 七、激励机制

### 7.1 虚拟成就

- 解锁徽章
- 称号（观鸟学徒→观鸟学者→观鸟专家→观鸟大师）
- 称号墙展示

### 7.2 社交分享

- 生成专属成就海报
- 分享到朋友圈
- 积分排行榜

## 八、技术实现

### 8.1 文件结构

```
miniprogram/
├── pages/
│   └── birding-school/          # 新增页面
│       ├── school-home/         # 学堂首页（地图）
│       ├── topic-detail/        # 主题详情页
│       └── level-detail/        # 关卡互动页
├── components/
│   ├── progress-map/            # 地图组件
│   ├── topic-card/              # 站点卡片组件
│   ├── level-card/              # 关卡卡片组件
│   ├── quiz选择题/              # 选择题组件
│   ├── quiz排序题/              # 排序题组件
│   └── quiz补全题/              # 补全题组件
└── utils/
    └── school-data.js           # 关卡数据
```

### 8.2 数据存储

- 用户进度存储在本地（wx.setStorageSync）
- 徽章解锁状态、积分、称号

### 8.3 资源需求

- 4个站点的手绘背景图
- 12张关卡封面（写实摄影）
- 16个徽章图标（手绘风格）
- 各题型的手绘元素

---

## 九、设计亮点

1. **地图式闯关** — 传递"探索自然"的旅程感
2. **手绘+摄影混搭** — 温暖感 + 沉浸感
3. **混合题型** — 保持用户兴趣，每关有惊喜
4. **线性解锁** — 确保价值观按逻辑顺序传递
5. **成就可视化** — 徽章墙 + 海报分享增强成就感
